## Transcript

### Assistant

Halo, ini Alex dari HDFC, menelepon terkait cicilan Bapak/Ibu yang akan jatuh tempo. Apakah ada waktu sebentar untuk bicara?

### User

Halo, iya ada waktu sebentar. Saya mau tanya soal cicilan yang jatuh tempo.

---

### Assistant

Boleh saya konfirmasi dulu atas nama siapa ya? Supaya saya bisa cek tanggal jatuh tempo dan jumlah cicilannya sesuai data Bapak/Ibu.

### User

Sekali bisa Anda kurangi?

---

### Assistant

Saat ini cicilan tidak bisa langsung dikurangi, tetapi ada program restrukturisasi seperti perpanjangan tenor atau pembayaran sebagian. Apakah Bapak/Ibu ingin informasi lebih lanjut atau ingin saya bantu mengajukan program tersebut?

### User

Bulan ini agak susah bayarannya nih. Ada solusi nggak?

---

### Assistant

Tentu. Bapak/Ibu dapat mengajukan penjadwalan ulang atau perpanjangan tenor agar cicilan bulanan menjadi lebih ringan. Apakah saya boleh membantu menghubungkan Anda ke tim kami untuk memprosesnya?

### User

Bisa saya bicara sama orang aslinya, bukan robot?

---

### Assistant

Baik. Saya akan menyambungkan Anda dengan tim kami yang dapat membantu secara langsung. Mohon ditunggu sebentar.

---

**Customer ended the call.**
